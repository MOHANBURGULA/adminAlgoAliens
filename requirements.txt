- Use: Loads the Poppins font used by the frontend UI.

- Use: Embeds the Monaco code editor in React pages.

- Use: Sends HTTP requests from the frontend to backend APIs.

- Use: Helps build reusable styled component variants.

- Use: Combines conditional CSS class names.

- Use: Adds animations and motion effects in the frontend.

- Use: Provides icon components used across the UI.

next
- Use: Runs the Next.js frontend framework.

- Use: Handles Google authentication and frontend session management.

radix-ui
- Use: Provides accessible UI primitives for frontend components.

- Use: Core library for building the frontend UI.

react-dom
- Use: Renders React components in the browser.

react-hot-toast
- Use: Shows toast notifications in the frontend.

recharts
- Use: Renders charts and dashboard visualizations.

tailwind-merge
- Use: Merges Tailwind CSS classes safely.

zustand
- Use: Manages lightweight frontend state.
