3. IMPLEMENTATION RULES FOR INTERNS
-----------------------------------

Never hardcode hex values in CSS or inline styles.
Wrong:
<div style={{ backgroundColor: '#131E32' }}>

Right:
<div style={{ backgroundColor: 'var(--bg-surface)' }}>

Use semantic variables.
Use --bg-base for the main app background or body.
Use --bg-surface for cards, modals, and panels.
Use --text-main for headings and primary data.
Use --text-muted for labels, subtext, and placeholders.

Applying Borders
Always apply border: var(--card-border) so the component adapts correctly across themes.

Backdrop Filters
If you are building a new floating element like a dropdown or sticky header, apply backdrop-filter: var(--glass-filter) so it inherits the blur effect when the Midnight Violet theme is active.
